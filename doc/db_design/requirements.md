# 简易问答社区 - 需求文档 (Day 1)

## 1. 项目概述
开发一个简易问答社区的后端服务，支持用户注册登录、发布问题、回答问题、发表评论等功能。

## 2. 核心功能模块

### 2.1 用户模块
- 用户注册（用户名、密码、邮箱）
- 用户登录（账号密码验证）
- 用户信息管理（基础信息存储）

### 2.2 问题模块
- 发布问题（标题、内容、标签、作者）
- 查看问题列表（分页）
- 查看问题详情

### 2.3 回答模块
- 针对问题发布回答
- 查看某问题的所有回答（分页）

### 2.4 评论模块
- 针对回答发表评论
- 查看某回答的所有评论（分页）

## 3. 核心实体
- User（用户）
- Question（问题）
- Answer（回答）
- Comment（评论）

## 4. 实体关系
- User 1:N Question（一个用户可发多个问题）
- User 1:N Answer（一个用户可写多个回答）
- User 1:N Comment（一个用户可写多条评论）
- Question 1:N Answer（一个问题可有多个回答）
- Answer 1:N Comment（一个回答可有多个评论）
